ABOUT DOOM

Doom is a first-person shooter game developed by id Software and released in 1993. It is considered one of the most significant and influential titles in video game history.

This browser version allows you to play the classic game without any downloads or installations. It includes the full first episode (Knee-Deep in the Dead) with all nine levels.

The game runs using WebAssembly technology, bringing the original DOS experience to modern browsers with improved controls and performance.
